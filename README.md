# 🎨 Intro Canvas 2D

Proyecto desarrollado con **HTML5 Canvas y JavaScript utilizando Programación Orientada a Objetos (OOP)**.

Este proyecto muestra cómo crear y renderizar objetos 2D dentro de diferentes canvas utilizando clases en JavaScript.

## 📌 Descripción del proyecto

La página contiene **tres canvas interactivos** que muestran diferentes formas de generar objetos gráficos.

### 1️⃣ Objeto 2D
Se crea un círculo centrado utilizando una clase `Circle`.

Características:

- Posición centrada
- Color aleatorio
- Texto dentro del círculo

---

### 2️⃣ Objeto con Random

Se genera un círculo con:

- Posición aleatoria
- Radio aleatorio
- Colores aleatorios
- Control para evitar que el círculo salga del canvas

---

### 3️⃣ Múltiples Objetos

Se generan múltiples círculos dentro del canvas:

- Cantidad controlada
- Posiciones aleatorias
- Colores aleatorios
- Cada círculo tiene un número identificador
